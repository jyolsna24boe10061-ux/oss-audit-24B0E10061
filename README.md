## What Each Script Does

### Script 1 – System Identity Report
This script displays the current user, home directory, Linux kernel version, system uptime, date, and a license message.

### Script 2 – Python Package Inspector
This script checks whether Python is installed and prints its version.

### Script 3 – Disk and Permission Auditor
This script checks some important Linux folders and displays their size and permission details.

### Script 4 – Log File Analyzer
This script reads a log file and counts how many times the word **error** appears.

### Script 5 – Open Source Manifesto Generator
This script asks the user a few simple questions and creates a small text file based on the answers.
